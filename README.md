# Smart Code Playground

🚀 **Smart Code Playground** is an AI-powered online code runner designed to help developers and students:

- Run JavaScript code directly in the browser
- Get **AI-generated hints**, suggestions, and explanations
- Practice coding with **daily challenges**
- Analyze **code quality** and difficulty level

---

## Features

- **Run Code:** Execute JavaScript code safely in the browser  
- **AI Hints:** Smart suggestions to improve your code  
- **Challenges:** Fun coding challenges to sharpen your skills  
- **Code Explanation:** Step-by-step breakdown of your code  
- **Copy Code:** Quickly copy your code snippet
